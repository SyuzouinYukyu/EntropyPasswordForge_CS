# AGENTS.md

This repository contains EntropyPasswordForge_CS.

- Keep all work within this repository.
- Describe this tool as an "OS CSPRNG + mouse-input entropy + CPU-jitter mixed" password generator.
- Do not claim that this tool is a strict physical TRNG.
- Do not collect, store, or mix keyboard input, key events, key timing, or modifier-key information as entropy.
- Do not output generated passwords to files, logs, Windows Event Log, telemetry, or network communication.
- When changing the application, keep the version number, executable name, README, CHANGELOG, window title, and project file aligned.
